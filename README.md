# Atomic Spectroscopy

Static GitHub Pages app for `Unit4_Section2_Atomic Spectroscopy`.

Built using `Lecture_App_Master_Prompt_and_Workflow.docx` as the governing workflow.

## Includes

- Lecture-sequence tabs based on slide ranges.
- Topic-specific interactive lab: Atomic Line Explorer, Atomization Source Path, Interference Decision Coach.
- Slide Studio with 45 slide images and accessible transcripts.
- Original lecture PDF download.
- Practice cards extracted from instructor-provided practice materials: 52 cards.
- No Word files exposed in the student-facing app.
- Keyboard navigation, visible focus, live feedback, text-first mode, graph summaries, and data tables.
